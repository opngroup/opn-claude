# opn-claude

> 오픈닥터 팀을 위한 Claude Code 커스텀 스킬 모음

## 스킬 목록

| 스킬                        | 파일                  | 설명                                                                                                         | 트리거 키워드                              |
| --------------------------- | --------------------- | ------------------------------------------------------------------------------------------------------------ | ------------------------------------------ |
| **Amplitude 이벤트 네이밍** | `amplitude-naming.md` | 기능 설명을 입력하면 오픈닥터 네이밍 규칙에 맞는 앰플리튜드 이벤트명과 프로퍼티를 제안하고, 노션에 자동 기록 | `앰플리튜드`, `amplitude`, `이벤트 네이밍` |
| **AI 코드 리뷰**            | `code-review.md`      | git diff 기반으로 변경된 코드를 분석하여 버그, 보안, 성능, 구조 이슈를 리뷰                                  | `코드 리뷰`, `code review`, `PR 리뷰`      |
| **PR 작성**                 | `write-pr.md`         | 현재 브랜치의 변경 사항을 분석하여 프로젝트 PR 템플릿에 맞는 PR 설명을 자동 생성                             | `pr 작성`, `write pr`                      |
| **AX 주간 리포트**          | `ax-report.md`        | 한 주간의 AI 활용 기록을 정리하여 슬랙으로 발송하는 주간 AX 리포트 생성                                      | `ax-report`, `주간 리포트`, `AX 리포트`    |
| **FSD 아키텍처 가이드**     | `fsd-architecture.md` | Feature Sliced Design 아키텍처 원칙에 따른 Next.js 프로젝트 구조 및 코딩 가이드                              | `FSD`, `아키텍처`, `디렉토리 구조`         |
| **토스 클린 코드**          | `toss-clean-code.md`  | 토스 Frontend Fundamentals 기반의 클린 코드 원칙 (가독성, 예측 가능성, 응집도, 결합도)                       | `클린 코드`, `토스 스타일`, `코드 리뷰`    |
| **CS 인터랙티브 튜터**      | `cs-tutor.md`         | 일상 비유 + 퀘스트 + 게임 진행으로 CS 개념을 체득시키는 튜터링 (네트워크, OS, 자료구조, DB 등)               | `CS 학습`, `CS 공부`, `stage`, `던전`      |

## 카테고리별 분류

### 개발 워크플로우

- **코드 리뷰** - 변경 코드의 품질 검수 자동화
- **PR 작성** - PR 설명 자동 생성

### 코드 품질

- **FSD 아키텍처 가이드** - 프로젝트 구조 및 레이어 규칙
- **토스 클린 코드** - 코드 작성 원칙 및 체크리스트

### 데이터/분석

- **Amplitude 이벤트 네이밍** - 이벤트 설계 + 노션 자동 기록

### 팀 협업

- **AX 주간 리포트** - AI 활용 기록 정리 및 슬랙 공유

### 학습

- **CS 인터랙티브 튜터** - 게임형 CS 개념 학습

## 사용 방법

이 레포의 `.md` 파일들을 Claude Code의 로컬 스킬로 등록하면, 트리거 키워드 입력 시 자동으로 해당 스킬이 활성화됩니다.
